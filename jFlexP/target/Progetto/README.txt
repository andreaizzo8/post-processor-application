Demo PostProcessor
IMPORTANTE NON SPOSTARE I FILE NELLA CARTELLA PROGETTO

Per l'utilizzo dell'applicazione fare doppio click su PostProcessor, se si vuole spostare sul Desktop o in un'altra cartella creare il collegamento di PostProcessor e spostare quello.
Tasto File: Caricamento del file aptsource.
Tasto Save: Servirà a definire la cartella di destinazione dell file di output
Tasto Run: Creazione del file di output, se non si determina la destinazione di output verrà salvato nella cartella principale Progetto.
Tasto Config: Inserimento dei valori Num Programma e Velocità Massima
Tasto Preview: Editor di testo per il file apt appena caricato.
